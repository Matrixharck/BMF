
# BMF TOURNOI - Formulaire complet avec serveur Node.js

## Étapes pour lancer le projet :
1. Ouvre un terminal dans ce dossier
2. Installe les dépendances :
   npm install express multer xlsx cors

3. Lance le serveur :
   node server.js

4. Ouvre index.html dans ton navigateur

## Fonctionnalités :
- Upload d'image de l'équipe et de la preuve de paiement
- Sauvegarde dans le dossier uploads/
- Génération automatique d'un fichier Excel (inscriptions.xlsx)
